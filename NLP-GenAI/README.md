# Medical RAG Question Answering

## Overview

Creates a retrieval augmented generation workflow over a medical manual to answer healthcare-related questions.

## Objective

Reduce information overload for healthcare users by retrieving relevant medical context and generating grounded responses.

## Domain

Generative AI / healthcare NLP

## Files

- Notebook: [`medical-rag-question-answering.ipynb`](./medical-rag-question-answering.ipynb)

Input files:

- `medical_diagnosis_manual.pdf`

## Key Techniques

- PDF ingestion
- Text chunking
- Embeddings/vector search
- RAG pipeline design
- LLM-based question answering

## How to Run

1. Open `medical-rag-question-answering.ipynb` in Jupyter Notebook, JupyterLab, or Google Colab.
2. Keep the input files in this same folder so the notebook can find them.
3. Run the notebook cells in order.
4. If the notebook installs packages, restart the kernel/runtime when prompted and continue running the remaining cells.

## Notes

This notebook was converted from an HTML export into a clean `.ipynb` file for GitHub. Outputs were removed during conversion to keep the notebook lightweight and easier to render.
