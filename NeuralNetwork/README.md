# ReneWind Generator Failure Prediction

## Overview

Develops neural network based models to predict wind turbine generator failures using transformed sensor data.

## Objective

Support predictive maintenance by identifying likely generator failures before they happen.

## Domain

Renewable energy / predictive maintenance

## Files

- Notebook: [`renewind-failure-prediction.ipynb`](./renewind-failure-prediction.ipynb)

Input files:

- `Train.csv`
- `Test.csv`

## Key Techniques

- Data preprocessing
- Neural network modeling
- Model tuning
- Train/test evaluation
- Failure prediction analysis

## How to Run

1. Open `renewind-failure-prediction.ipynb` in Jupyter Notebook, JupyterLab, or Google Colab.
2. Keep the input files in this same folder so the notebook can find them.
3. Run the notebook cells in order.
4. If the notebook installs packages, restart the kernel/runtime when prompted and continue running the remaining cells.

## Notes

This notebook was converted from an HTML export into a clean `.ipynb` file for GitHub. Outputs were removed during conversion to keep the notebook lightweight and easier to render.
