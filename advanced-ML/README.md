# EasyVisa Approval Prediction

## Overview

Uses ensemble learning and model tuning to predict visa application outcomes from employer and applicant case data.

## Objective

Help prioritize visa applications and identify factors associated with certification or denial decisions.

## Domain

Immigration / workforce analytics

## Files

- Notebook: [`easyvisa-approval-prediction.ipynb`](./easyvisa-approval-prediction.ipynb)

Input files:

- `EasyVisa.csv`

## Key Techniques

- Exploratory data analysis
- Feature engineering
- Ensemble techniques
- Hyperparameter tuning
- Model comparison

## How to Run

1. Open `easyvisa-approval-prediction.ipynb` in Jupyter Notebook, JupyterLab, or Google Colab.
2. Keep the input files in this same folder so the notebook can find them.
3. Run the notebook cells in order.
4. If the notebook installs packages, restart the kernel/runtime when prompted and continue running the remaining cells.

## Notes

This notebook was converted from an HTML export into a clean `.ipynb` file for GitHub. Outputs were removed during conversion to keep the notebook lightweight and easier to render.
