# Personal Loan Campaign Prediction

## Overview

Builds a supervised machine learning model for AllLife Bank to predict which liability customers are likely to purchase a personal loan.

## Objective

Identify high-propensity customers, understand important customer attributes, and support more targeted marketing campaigns.

## Domain

Banking / marketing analytics

## Files

- Notebook: [`personal-loan-prediction.ipynb`](./personal-loan-prediction.ipynb)

Input files:

- `Loan_Modelling.csv`

## Key Techniques

- Exploratory data analysis
- Data preprocessing
- Classification modeling
- Model evaluation
- Feature importance and customer targeting insights

## How to Run

1. Open `personal-loan-prediction.ipynb` in Jupyter Notebook, JupyterLab, or Google Colab.
2. Keep the input files in this same folder so the notebook can find them.
3. Run the notebook cells in order.
4. If the notebook installs packages, restart the kernel/runtime when prompted and continue running the remaining cells.

## Notes

This notebook was converted from an HTML export into a clean `.ipynb` file for GitHub. Outputs were removed during conversion to keep the notebook lightweight and easier to render.
