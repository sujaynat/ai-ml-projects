# Helmet Detection Computer Vision Model

## Overview

Builds an image classification model to identify whether workers are wearing safety helmets in workplace images.

## Objective

Classify images into helmet and no-helmet categories to support workplace safety monitoring.

## Domain

Computer vision / workplace safety

## Files

- Notebook: [`helmet-detection.ipynb`](./helmet-detection.ipynb)

Input files:

- `Labels_proj.csv`
- `images_proj.npy`

## Key Techniques

- Image data loading
- Computer vision preprocessing
- Deep learning classification
- Model evaluation
- Safety compliance use case analysis

## How to Run

1. Open `helmet-detection.ipynb` in Jupyter Notebook, JupyterLab, or Google Colab.
2. Keep the input files in this same folder so the notebook can find them.
3. Run the notebook cells in order.
4. If the notebook installs packages, restart the kernel/runtime when prompted and continue running the remaining cells.

## Notes

This notebook was converted from an HTML export into a clean `.ipynb` file for GitHub. Outputs were removed during conversion to keep the notebook lightweight and easier to render.
