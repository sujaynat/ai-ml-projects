# SuperKart Sales Forecasting and Deployment

## Overview

Builds a sales prediction model for SuperKart using product and store attributes, with model deployment-oriented workflow steps.

## Objective

Forecast product-store sales to support inventory planning, regional strategy, and retail decision-making.

## Domain

Retail analytics / model deployment

## Files

- Notebook: [`superkart-sales-forecasting.ipynb`](./superkart-sales-forecasting.ipynb)

Input files:

- `SuperKart.csv`

## Key Techniques

- Exploratory data analysis
- Regression modeling
- Model evaluation
- Model serialization/deployment workflow
- Retail sales insights

## How to Run

1. Open `superkart-sales-forecasting.ipynb` in Jupyter Notebook, JupyterLab, or Google Colab.
2. Keep the input files in this same folder so the notebook can find them.
3. Run the notebook cells in order.
4. If the notebook installs packages, restart the kernel/runtime when prompted and continue running the remaining cells.

## Notes

This notebook was converted from an HTML export into a clean `.ipynb` file for GitHub. Outputs were removed during conversion to keep the notebook lightweight and easier to render.
